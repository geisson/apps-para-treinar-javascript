Visualizador de Border Radius
Tier 1: iniciante

A propriedade do raio de fronteira pode ter vários valores alterados. Visualize como a forma fica ao alterar esses valores.

Ações de usuário

  • O usuário pode ver uma caixa que tem uma propriedade aplicada a elaborder-radius
  • O usuário pode alterar os 4 valores que são aplicados na caixa (superior esquerdo, superior direito, inferior esquerdo, inferior esquerdo, inferior direito)border-radius
  • O usuário pode copiar o CSS resultante para a área de transferência
